# Smart Expense Tracker

A production-ready personal finance management web application built with Flask and Vanilla JavaScript. Track your income, expenses, and savings with an intuitive dashboard, charts, and smart financial advice.

## 🌟 Features

### Core Features
- **Complete CRUD Operations** - Create, Read, Update, Delete transactions
- **Smart Dashboard** - Real-time balance, income, expense, and savings tracking
- **Budget Planner** - Monthly budget with progress tracking and warnings
- **Financial Analysis** - Automatic calculation of financial metrics and insights
- **Smart Advice** - Personalized financial tips based on your spending patterns
- **Emergency Fund Calculator** - Track and plan your emergency savings

### Data Management
- **Search & Filter** - Search by title/notes, filter by category/type/month
- **Sorting** - Sort by newest, oldest, highest, or lowest amount
- **Import/Export** - JSON import and export functionality
- **Print Reports** - Print-friendly financial reports

### Visualizations
- **Expense by Category** - Interactive pie chart
- **Income vs Expense** - Bar chart comparison
- **Budget Progress** - Visual progress bar
- **All charts update automatically** with every transaction change

### User Experience
- **Light/Dark Theme** - Toggle between themes with local storage persistence
- **Mobile First** - Fully responsive design optimized for mobile users
- **Touch Friendly** - Large buttons and touch targets for mobile
- **Toast Notifications** - Real-time feedback for all actions
- **Confirmation Dialogs** - Prevent accidental deletions
- **Loading Spinners** - Visual feedback during API calls

### Security & Validation
- Input validation on both client and server sides
- Proper HTTP status codes
- Error handling with meaningful messages
- Duplicate submission prevention

## 🛠️ Technology Stack

### Frontend
- HTML5
- CSS3 (with Tailwind CSS)
- Vanilla JavaScript (ES6+)
- Chart.js for data visualization
- Font Awesome for icons
- Google Fonts (Inter)

### Backend
- Python (Flask)
- RESTful API architecture
- JSON file-based storage (No database required)
